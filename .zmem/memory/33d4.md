# Power Eagle is built with React 18, TypeScript, Vite, TailwindCSS, DaisyUI, the Eagle extension environment, and Node modules exposed by Eagle.

## Context
- The project combines a modern frontend stack with host-specific Eagle capabilities for filesystem and process access.
- The technical notes emphasize both the web UI technologies and the Eagle runtime environment.

## Content
- The frontend is built with React 18, TypeScript, Vite, TailwindCSS, and DaisyUI.
- Runtime integrations rely on Eagle's extension environment, including the global Eagle API and Node modules such as fs, path, and child_process.
- Browser-side plugin UI work happens inside Eagle's Electron-based environment rather than a generic browser tab.

## Consequences / notes
- Stack changes need to account for both web UI constraints and host-environment capabilities.
- The project is not a purely browser-only app; host integration is part of the technical baseline.
