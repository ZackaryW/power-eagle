# Python callback support depends on the py-eagle-cooler library for Python-side EagleCallback signal emission.

## Context
- Python plugins need a convenient author-facing API for triggering Eagle operations from Python code.
- The callback architecture depends on a cooperating Python package rather than forcing plugin authors to emit raw protocol strings manually.

## Content
- The py-eagle-cooler library provides the EagleCallback interface used by Python plugins.
- That library emits the callback signals consumed by the Power Eagle evaluator.
- The dependency is external to this repository but central to the Python callback story.

## Consequences / notes
- Compatibility with py-eagle-cooler matters for Python plugin ergonomics.
- Changes to the callback protocol need coordination with the Python-side library contract.
